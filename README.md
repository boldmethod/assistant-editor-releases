# Assistant Editor — downloads

Public host for the **Assistant Editor** Mac app: signed/notarized `.dmg` releases and the
Sparkle auto-update feed (`appcast.xml`). The app source lives in a separate private repo.

## Install

Download the latest `AssistantEditor-<version>.dmg` from
[Releases](https://github.com/boldmethod/assistant-editor-releases/releases/latest), open it,
and drag the app to Applications. Requires an Apple Silicon Mac on macOS 15+. The app updates
itself after that.
